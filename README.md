# China Travel Guide

A comprehensive travel guide website for foreign visitors to China. Built with Hexo.

## Website
🌐 **Live Site**: https://qzw-alt.github.io/china-travel-guide

## Features

- 📚 Comprehensive travel guides
- 🗺️ City itineraries and recommendations
- 📝 Visa application guides
- 🚄 Transportation tips
- 💡 Practical advice for first-time visitors

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- Git

### Local Development

```bash
# Install dependencies
npm install

# Start local server
npx hexo server

# Generate static files
npx hexo generate
```

### Create New Post

```bash
npx hexo new post "Your Post Title"
```

### Deploy to GitHub Pages

```bash
# Install deployer
npm install hexo-deployer-git --save

# Deploy
npx hexo deploy
```
